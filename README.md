# My Trading Chart

A modern trading chart application built with React, TypeScript, and Vite.

## Features

- Interactive trading charts using Lightweight Charts
- Dark/Light theme support
- Modern UI components with Radix UI and Shadcn
- Responsive design with Tailwind CSS

## Tech Stack

- **Framework**: React 18 with TypeScript
- **Build Tool**: Vite
- **UI Components**: Radix UI, Shadcn
- **Styling**: Tailwind CSS
- **Data Visualization**: Lightweight Charts, Recharts
- **State Management**: React Query
- **Form Handling**: React Hook Form
- **Routing**: React Router DOM

## Getting Started

### Prerequisites

- Node.js (Recommended version: 18.x or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone <YOUR_REPOSITORY_URL>
cd my-trading-chart
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Start the development server
```bash
npm run dev
# or
yarn dev
```

The application will be available at `http://localhost:5173`

## Development

- Run the development server: `npm run dev`
- Build for production: `npm run build`
- Preview the production build: `npm run preview`
- Lint the code: `npm run lint`
